## Setup
Frontend - React\
Backend - Python 

## Configure React-Vite client
```powershell
cd client
npm install
```

## Configure Python-Flask server
```powershell
cd server
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Run Vite client
```powershell
cd client
npm run dev
```

## Run Flask server
```powershell
cd server
venv\Scripts\activate
python src/main.py
```
